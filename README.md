# Mac Monitor

一个用于监控局域网 Mac 电脑运行状态的 iPhone Dashboard 应用。

## 项目结构

```
mac-monitor/
├── iOS/          # iPhone Dashboard 应用
└── agent/        # Mac 监控 Agent 服务
```

## 正在开发中...

项目正在初始化，完整代码即将添加。

## 技术栈

- **iOS App**: SwiftUI, Combine, Bonjour
- **Mac Agent**: Swift (Hummingbird) / Python (FastAPI)
